Login Not Available from Browser

This server does not support direct username/password login from the browser.

microauthd enforces strict token security. To authenticate, you must use:

  • A server-side login flow using madAuthClient (ASP.NET Core)
  • A custom SPA or mobile app that calls /token securely from your backend
  • Machine-to-machine authentication using the client_credentials grant

Interactive browser-based flows (like PKCE) are not yet supported.

See the documentation or examples for implementation guidance.
